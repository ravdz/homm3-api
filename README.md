# HoMM3 api

A RESTful API for Heroes of Might & Magic 3

## Tech Stack

- NestJS
- TypeScript
- PostgreSQL

## Installation

```bash
npm install
```

## Environment variables
Copy .env.example file and fill it.

```bash
cp .env.example .env.development
```

## Running the app

```bash
# development & watch mode
npm run start:dev

# production mode
npm run start:prod
```
